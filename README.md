# react-native-readdirp

Recursive, React Native version of fs.readdirp.

## Installation

```sh
npm install react-native-readdirp
```

## Usage

```js
import Readdirp from "react-native-readdirp";

// ...

const result = await Readdirp.multiply(3, 7);
```
